| 原对象 | 拷贝方法    | 拷贝方式 | 生成的对象 |
| :----- | :---------- | -------- | ---------- |
| 不可变 | copy        | 浅拷贝   | 不可变     |
| 不可变 | mutableCopy | 深拷贝   | 可变       |
| 可变   | copy        | 深拷贝   | 不可变     |
| 可变   | mutableCopy | 深拷贝   | 可变       |

用copy修饰的或者赋值的变量肯定是不可变的.

用copy赋值,要看源对象是否是可变的,来决定只拷贝指针,还是也拷贝对象到另一块内存空间.

### NSString用copy修饰

当赋值为NSString的值是可变值时,copy进行了深拷贝,值变化NSString不跟随改变.

而使用string修饰的NSString因为是指针复制,跟随值改变.
